# Licensing Strategy Note v1.1

This pack is an overlay for deciding and documenting the licensing posture of `andyai-skill-engine`.

## Recommended default

**Hybrid / open-core**
- public repo for the core
- commercial or source-available treatment for advanced enterprise layers

## Why

The repo is strong enough to benefit from:
- public credibility
- GitHub discoverability
- community validation

But the future monetization should sit above the core:
- enterprise governance
- hosted trust tooling
- premium exporters
- managed services
