# PUBLIC / PRIVATE SETUP PLAN v1.1

## Recommended GitHub structure

### Public repo
`andyai-skill-engine`

Contains:
- public architecture
- public docs
- public schema
- public examples
- public validator + CLI baseline

### Later private or protected repos
Examples:
- `andyai-skill-engine-enterprise`
- `andyai-trust-cloud`
- `andyai-copilot-skill-bridge`
- `andyai-skill-engine-commercial-packs`

## Why split like this

The public repo becomes:
- credibility layer
- onboarding layer
- developer trust layer

The protected repos become:
- monetization layer
- enterprise differentiation layer
- partner-only distribution layer
