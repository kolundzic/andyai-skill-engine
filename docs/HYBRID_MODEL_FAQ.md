# Hybrid Model FAQ v1.1

## Is hybrid weaker than full open source?
Not necessarily. It often gives a better business path while still allowing public credibility.

## Is Fair Source the same as OSI open source?
No. It is public/readable and intentionally producer-protective, but it is not the same category as standard OSI-approved open-source licensing.

## What should stay public here?
The core engine baseline.

## What should stay commercial?
Anything that becomes the main operational moat:
- enterprise trust automation
- premium adapters
- hosted management
- support/compliance layers
