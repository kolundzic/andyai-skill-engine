# Apache-2.0 Transition Notes v1.0

## Why Apache-2.0

Apache-2.0 is a standard permissive open-source license widely recognized by developers and enterprises.

## Practical effect

People can use, modify, and redistribute the code under Apache-2.0 terms.

## Important note

This protects the public core through a standard legal framework, but it does not prevent competitors from building around the public codebase.

That is why monetization should come from:
- enterprise packs
- managed services
- support
- advanced adapters
- hosted trust tooling
