# Public Core Policy v1.0

## Repo posture

`andyai-skill-engine` is the public core.

## Public core includes

- skill schema
- examples
- validator core
- CLI baseline
- CI/test baseline
- public documentation

## Commercial layer belongs elsewhere

Future monetization should live above this repo, for example:
- enterprise governance packs
- hosted trust validation
- advanced exporters
- premium adapters
- support / SLA layers

## Rule

Do not bury the monetization moat inside the public baseline repo.
Build the moat in separate higher-order repos, services, or enterprise packs.
