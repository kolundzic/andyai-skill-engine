# Repo Positioning v0.1

## Positioning Statement

AndyAI Skill Engine is the master repository for skill-based orchestration architecture.

## What It Is

- a reusable skill system
- a foundation for export adapters
- a bridge between execution and governance

## What It Is Not

- not only a Copilot repo
- not only a prompt pack
- not only a schema collection

## Canonical Position

Compatibility paths may multiply over time, but the master logic stays here.
