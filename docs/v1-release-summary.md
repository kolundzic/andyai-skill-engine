# v1 Release Summary

`v1.0` is the first public baseline of AndyAI Skill Engine.

## What is stable enough to present

- repo purpose
- schema + examples
- validator
- CLI entrypoint
- trust-aware packaging docs
- CI and tests
- public onboarding docs

## What is intentionally still light

- semantic validation depth
- number of skill examples
- exporter breadth
- release artifact sophistication

## Meaning of v1.0 here

This is a **public baseline release**, not the end-state of the project.
